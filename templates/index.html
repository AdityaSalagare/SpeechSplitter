What is Speech Diarization?

Speech diarization is the process of partitioning an audio stream into segments according to speaker identity. In simpler terms, it answers the question: "Who spoke when?" This technology is essential for transcribing meetings, interviews, and conversations where multiple speakers are involved.

Key Features

Voice Activity Detection

Automatically identifies when someone is speaking and filters out silence and noise.

Speaker Separation

Distinguishes between different speakers using MFCC (Mel-frequency cepstral coefficients) features.

Easy Integration

Simple API endpoints for both React and Flask applications. Drop-in components available.

Audio Export

Download separate audio files for each speaker identified in the conversation.

How It Works

  1. Audio is recorded or uploaded to the system
  2. Voice Activity Detection (VAD) identifies speech segments
  3. MFCC features are extracted from speech segments
  4. Clustering algorithms separate speakers based on voice characteristics
  5. Individual audio segments are created for each speaker

Quick Integration Guide

React

Add the Speech Diarizer component to your React app:

import { SpeechDiarizer } from 'speech-diarizer';

function MyComponent() {
  return (
    <SpeechDiarizer 
      apiEndpoint="/api/webrtc"
      onDiarized={(result) => console.log(result)}
    />
  );
}

Flask

Add the diarization blueprint to your Flask app:

from flask import Flask
from api.routes import api_bp

app = Flask(__name__)
app.register_blueprint(api_bp, url_prefix='/api')

if __name__ == '__main__':
    app.run(host='0.0.0.0', port=5000)
